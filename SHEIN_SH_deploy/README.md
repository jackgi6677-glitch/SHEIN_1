# SHEIN 批量上传模板工具

配置 Excel 模板与飞书字段映射，按货号拉取飞书数据并生成上传文件。

---

## 上架到网页（推荐：同事免安装）

若希望同事**不用装 Python、直接浏览器打开链接就用**，可把本工具部署到网页。  
详见 **`上架说明.md`**：用 GitHub + Streamlit 官方免费托管，部署后把链接发给同事即可。

---

## 发给同事使用（本地运行）

### 推荐：用「打包发给同事」脚本（干净版 + 自动安装）

1. **你这边**：双击运行 **`打包发给同事.bat`**，会在当前目录生成 **`SHEIN_SH_发给同事.zip`**（只含运行所需文件，不含你的个人模板与配置）。
2. 把该 ZIP 发给同事（微信/钉钉/网盘均可）。
3. **同事那边**：解压后双击 **`安装并启动.bat`**：
   - 若未安装 Python，会提示并打开官网，安装时**务必勾选「Add Python to PATH」**，安装后关闭窗口再重新双击本脚本；
   - 依赖会自动检测并安装；
   - 然后自动启动应用，浏览器打开 `http://localhost:8501`。
   - **若点击没反应或报错**：请打开解压后的 **`环境配置说明.txt`** 按步骤排查。

### 方式二：手动打包整个文件夹

把整个项目文件夹打成 ZIP 发给同事，让对方解压后先按下面「环境准备」安装 Python 和依赖，再双击 `安装并启动.bat` 或 `启动应用.bat`。

---

## 环境准备（同事电脑需做一次）

1. **安装 Python 3.8+**  
   - 打开 [python.org](https://www.python.org/downloads/) 下载并安装，勾选「Add Python to PATH」。
2. **安装依赖**  
   - 在项目文件夹里打开终端（或在该目录下 Shift+右键 → “在此处打开 PowerShell”），执行：
   ```bash
   pip install -r requirements.txt
   ```
3. **（可选）飞书配置**  
   - 使用前需有飞书应用凭证：App ID、App Secret、App Token、Table ID，在应用内「飞书环境配置」中填写并测试连接。

---

## 启动应用

- **首次使用 / 不确定是否装好环境**：双击 **`安装并启动.bat`**（会先检测 Python 与依赖，缺失则提示或自动安装，再启动应用）。
- **已确认环境没问题**：双击 **`启动应用.bat`** 即可。

或在项目目录下打开终端，执行：

```bash
python -m streamlit run app.py  
```

浏览器会自动打开，地址：**http://localhost:8501**（若未自动打开，可手动输入该地址）。

---

## 使用步骤简述

- **第一步：模板配置与保存**  
  上传 SHEIN 官方 Excel 模板 → 配置「规格与固定属性」、「飞书字段绑定」等 → 保存为模板（会同时保存 Excel 结构，第二步可不再上传）。

- **第二步：生成上传文件**  
  选择已保存的模板并加载 → 输入卖家 SKU、供方货号等 → 在「核心动态录入区」填写/核对信息 → 点击「拉取飞书并生成 Excel」→ 下载生成的 Excel。

首次使用建议先完成第一步并保存模板，之后可直接在第二步选模板生成，无需重复上传 Excel。
